# homebrew-kubepyrometer

Homebrew tap for [KubePyrometer](https://github.com/spectronauts/KubePyrometer) — Kubernetes control-plane load-testing harness.

## Install

```bash
brew tap spectronauts/kubepyrometer
brew install kubepyrometer
```

## Usage

```bash
kubepyrometer init       # generate a config file with all defaults
kubepyrometer run        # run the load test
kubepyrometer help       # see all commands
```

See the [KubePyrometer README](https://github.com/spectronauts/KubePyrometer#readme) for full documentation.
